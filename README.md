# EngineeringSolver

I developed this desktop app for my CSE 104 class using Java. It's designed to solve core engineering problems in Calculus II, Physics II, and Discrete Mathematics with a step-by-step educational approach.


## Technical Highlights

OOP Based: I used an abstract class structure to make the code modular and easy to expand with new subjects.  
Java Swing UI: Built an interactive interface where you can pick a subject and see the solution instantly.  
Step-by-Step Logic: The app doesn't just give the answer; it shows the formula, the substitution, and the final result so you can learn while you solve.  
Precision Formatting: I wrote a custom algorithm to display small constants (like $10^{-12}$) and engineering units in a clean, textbook-compliant way.  

## What's Inside?
Physics II: Resistance, Capacitance, Magnetic Fields, and Transformers.  
Calculus II: Integration rules (Power Rule, Trig), Area, and Arc Length.  
Discrete Math: Permutations, Combinations, and Logic gates.
